HTML

    HTML é uma linguagem de marcação utilizada para desenvolver sites 
    para a web. É composto por elementos definidos por meio de tags e 
    suporta estilização e scripts externos como por exemplo: Java Script.

CSS

    CSS é a sigla para Cascade Style Sheets ou folha d eestilos em cascata.
    É utilizado para estilizar componnentes de uma página HTML.
    Pode ser utilizado dentro da tag ou fora dela, em uma tag ou até mesmo em um arquivo externo.
